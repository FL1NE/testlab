# simple-nodejs

## What?

Did these things.

- `$ npm init`
- Delete `main` line from package.json because we don't use this code from outside.
- Add `start` script to `package.json`
- Add `random` module from npm
- Integrate webpack for build

## How?

### DEBUG

```
$ npm run dev
```

### PRODUCTION

```
$ npm run build
$ npm run start
```
